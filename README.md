# cnode社区uni-app版

cnode社区uni-app版

> 数据api来源cnode公开api,地址： https://cnodejs.org/api
>
> axios风格的request封装改造自 [SnoopyChen](https://ext.dcloud.net.cn/publisher?id=1260)的`封装的request网络请求`：https://ext.dcloud.net.cn/plugin?id=159



## 特性

- Api统一管理（支持请求响应拦截器）
- 兼容3个端
- 覆盖全局的错误提示
- uni-app内 npm 包安装使用
- 下拉刷新
- 友好的体验
